# 汉字学习网站

一个专为成人初学者设计的现代化汉字学习网站，提供美观的界面、互动式笔顺动画、语音朗读和学习进度跟踪。

## 🎯 项目概述

这是一个基于纯静态技术（HTML5、CSS3、JavaScript）的汉字学习平台，专为成人初学者设计。网站提供优雅的学习界面，包含笔顺动画、语音朗读、学习进度管理等功能，并支持轻松扩展汉字库。

## ✨ 功能特性

### 核心功能
- **汉字展示**：清晰展示汉字、拼音（带声调）和常用词组
- **笔顺动画**：使用 Hanzi-writer 库展示汉字书写笔顺
- **语音跟读**：基于浏览器 Web Speech API 的汉字和词组发音
- **进度管理**：自动保存学习进度，支持重置
- **响应式设计**：完美适配手机、平板和桌面设备

### 扩展功能
- **复习模式**：随机复习已学汉字
- **学习统计**：可视化展示已学字数、学习时间等数据
- **个性化设置**：可调整笔顺速度、语音语速、主题颜色
- **字库管理**：通过管理面板轻松添加、编辑汉字
